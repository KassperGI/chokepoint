# CHOKEPOINT -_-

>>> BRUTAL HARDWARE BOTTLENECK DIAGNOSTIC <<<

🔗 **LIVE PROTOCOL:** [Launch Diagnostic App](https://chokepoint-ludhmucqmguhstm9xpw5yq.streamlit.app/#chokepoint)

A stark, minimalist hardware diagnostic tool powered by LLMs. It evaluates PC component configurations against specific workloads and market prices, delivering cold, calculated, and slightly aggressive system analysis.

### >>> OPERATIONAL FEATURES
* **PERFORMANCE DIAGNOSTIC:** Pinpoints exact system bottlenecks with calculated precision.
* **REQUIRED OPTIMIZATION:** Provides a prioritized, practical component upgrade path.
* **FINANCIAL AUDIT:** Evaluates user expenditure against current market realities to detect financial inefficiency.
* **FAIR ACQUISITION VALUE:** Generates a dynamic, fair market price recommendation for the exact hardware loadout.
* **STARK UI:** Custom CSS implementation featuring a modular, dot-matrix, "Nothing Tech" authoritarian aesthetic.

### >>> SYSTEM ARCHITECTURE
* **Frontend/UI:** Streamlit (Python) with Custom HTML/CSS Injection
* **Intelligence Engine:** Google Gemini API (`gemini-2.5-flash`)
* **Typography:** `Major Mono Display` & `Cutive Mono`
